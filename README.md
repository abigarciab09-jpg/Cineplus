# Cineplus
Aplicación de cine que consta de 4 pantallas ( inicio de sesión , taquilla , dulceria y ticket )


Ventana 1 :  Inicio de sesión 
-se crea  la pantalla de inicio de sesión 
-Se hará con datos ya preestablecidos en el código los cuales serán ( Abi como usuario y 1234 como contraseña) si no están esos datos no podrán ingresar a las demás ventanas.

Ventana 2 : Taquilla
-se crea la pantalla de la venta de boletos
- se escogerá  la película, la cantidad de boletos si son de adulto o de niño  , horario 
- al finalizar la selección se le mostrará un resumen de boletos adquiridos y de especificaciones de las opciones seleccionadas
-se creara botones para limpiar o siguiente 
-al seleccionar el botón limpiar se eliminarán las opciones elegidas 
-si se seleccióna el botón siguiente se mostrará el resumen de taquilla y aparecerá un si o no al seleccionar si se enviara a la ventana 3 (dulcería ) y al presionar no se cancelara compra

Ventana 3 : Dulcería 
- Se crea la pantalla de Dulcería 
-donde aparecerán productos como refrescos palomitas y combos 
-podras escoger todos los productos que deses 
-ahi se pondrá el subtotal de productos seleccionados
-se pondrán botones 
-al seleccionar siguiente se pondrá el resumen de taquilla y si no se limpiara la pantalla 

Ventana 4 : Ticket 
-Se creara la ventana de ticket
-Aprecera el resumen de taquilla y de dulcería 
-Aprecera el subtotal de taquilla y dulcería 
-Tendra el total (se sumarán los dos subtotales de taquilla y de dulcería) 
- se pondrá botones para seleccionar si deseas realizar la compra y finalizar el programa o si quieres regresar a las otras pantallas
